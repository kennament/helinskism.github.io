Default Dictionary

The default dictionary is a massive collection of linguistic patterns.

The default dictionary was developed based on David Kaufer and Brian Butler's earlier theoretical work in rhetoric and their applied work in representational theories of language.

When completed and ready for export, dictionary files are compiled and exported into a .DAT file. DocuScope cannot edit these .DAT files, which means the default dictionary cannot be edited. If desired, you can create your own dictionary to use in analysis.
