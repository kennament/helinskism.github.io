DocuScope Overview

Use DocuScope to analyze either a single text or a group corpus.

The following process outlines the general tasks involved in analyzing a document.

  1. Open DocuScope
  2. Load a Dictionary
  3. Load either a single text or a corpus
  4. Use filtering criteria to examine the text

DocuScope offers more features to examine a corpus, including creating a CSV file of the results of the analysis. Learn how to set up DocuScope, then jump into loading your first text.
