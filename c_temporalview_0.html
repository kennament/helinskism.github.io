Temporal View

The lower section of the DocuScope application displays the temporal view, which shows you how language patterns are used over time (from the beginning to the end of the text).

Use the following tools to visualize rhetorical patterns over time.

  • Highlight a Paragraph
    The Temporal View displays how rhetorical patterns are used in each paragraph. Highlight a paragraph to focus in on a certain passage.
  • Graphing Options
    The Temporal View offers several ways to view rhetorical patterns.
  • Display Multiple Clusters
    Visualize multiple clusters in the Temporal View.
