Add Clusters and Dimensions

Add a new cluster or dimension to your dictionary.

After opening the dictionary in the editor, cluster and dimension controls are in the top left part of the application window.
  1. Select Add...
    • Add cluster
    • Add dimension
      Note: You must select a cluster first before adding a dimension
  2. In the Properties window below, enter a name for the cluster / dimension in the Name Text Field and click Update
If you make a mistake, select the cluster or dimension and hit Delete to remove the desired category.
