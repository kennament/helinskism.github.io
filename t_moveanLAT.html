Move an LAT

Move an LAT to another dimension.

Once you have your clusters and dimensions set up, use the Move command in the LAT window.

  1. Select the desired LAT in the LAT Window
  2. Select Move
  3. In the Pop Up Menu, select the destination dimension and click OK
