Dictionaries

Dictionaries are the key to making DocuScope work.

Dictionaries are files that house the rhetorical features and patterns used to analyze a text. Every instance of DocuScope requires a dictionary to be loaded before any analysis can be done.

The default dictionary, developed by members of the Carnegie Mellon Department of English, currently holds over 40 million linguistic patterns of English, which are classified into over 100 categories of rhetorical effects.

You can load this default dictionary, or if desired, you can create and edit your own. DocuScope allows analysts to systematically explore and create a new domain-specific dictionary for them to use on either a single text or a corpus.
