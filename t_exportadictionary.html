Export a Dictionary

Finalize and create a .DAT file of the dictionary.

The .DAT file contains all the dictionary pattern files and cannot be edited. To export the dictionary
  1. Click the Menu Icon and select Export to DAT..
  2. Fill out information about the dictionary in the dialog boxes
  3. Press Create
