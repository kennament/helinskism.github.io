Graphing Options

The Temporal View offers several ways to view rhetorical patterns.

The Temporal View graphs display the selected clusters and corresponding dimensions based on four criteria. Change the graphing results between the following.
Option Definition
Count Displays the number of instances of a pattern in each paragraph
Frequency Displays the number of instances of a pattern relative to the length of the paragraph
Log Scale Displays the number of instances of a pattern over a wide range of values
Binary Displays a graph that indicates only if a rhetorical pattern is present or not. (1 = Yes, 0 = No)
