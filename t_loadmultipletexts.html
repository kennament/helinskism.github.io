Load a Single Group of Texts

Upload a single set of multiple texts to be analyzed together.

Use the file explorer on your device to set up the folder structure, then import into DocuScope for analysis.
  1. Create a new folder for your project
  2. Place all the text files to be analyzed in this folder
    Do not include sub-folders, just the text files to be analyzed
  3. Launch DocuScope and Load a Dictionary
  4. Select Analysis > Generate CSV from Corpus
  5. Navigate to the Corpus folder created in Step 1 and hit Select
    An output folder will automatically be created in the folder that contains your corpus folder. A CSV file is generated for each level of the dictionary.
