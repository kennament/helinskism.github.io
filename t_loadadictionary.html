Load a Dictionary

Before analyzing a text, you need to load a DocuScope dictionary.

The Default Dictionary is included with DocuScope as a .DAT file.
  1. Open DocuScope
  2. In the Menu, click Dictionary > Load from DAT-
  3. In the folder view, select the .DAT file
    Note: Loading a dictionary usually takes about 2-3 minutes, but may take longer on computers with limited memory size.
