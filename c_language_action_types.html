Language Action Types

Language Action Types (LATs) are the smallest level of categorization.

LATs are usually defined by individual words, or a smaller string of words than clusters and dimensions. They differentiate Dimensions

An example of a Dimension would be
  • First Person Interior

    Which contains the LATs

  • Self-Disclosure
  • Self-Disclosure Interior
  • Autobio
