Visualizing Rhetoric

Once a text is loaded, visualize the rhetorical features used by the author.

The loaded dictionary contains hundreds of rhetorical filters with different levels of scrutiny. In many cases, it helps to first filter the results to exclude categories with no results. You can then select different categories to learn more about the rhetorical definition behind it. Use the Temporal View section to add further layers of analysis.

  • Filter Results
    When analyzing a single text, filter results by either Cluster or Dimension
  • Choose a Cluster
    Pick a rhetorical cluster to visualize results.
