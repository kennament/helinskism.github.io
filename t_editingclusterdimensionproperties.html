Edit Cluster and Dimension Properties

Modify existing clusters and dimensions to add descriptions and labels.

After opening the dictionary in the editor window, make edits to the selected cluster or dimension with the properties section in the bottom left.
Note: There are three properties to edit. See Dictionary Properties for more information about these properties.
  • Name
  • Label
  • Content
  1. Select the desired cluster / dimension in the Clusters / Dimensions Window
  2. In the Properties Window below, change the name, label, and add a definition of the cluster / dimension
  3. Hit Update to save your changes
During text analysis, these properties are valuable for users to know what a cluster / dimension is. Updating the content section gives users context to the rhetorical pattern during text analysis.
