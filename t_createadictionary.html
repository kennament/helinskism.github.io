Create A Dictionary

Create a new dictionary to use in DocuScope.

To create a new dictionary file
  1. Select Dictionary > New..
    This will open a dialog for creating the dictionary
  2. Select a folder where you would like to save the dictionary
  3. Enter a unique name and hit Create
