Creating a New Dictionary

Create your own custom dictionary to use in DocuScope.

DocuScope allows analysts to explore and create a new domain-specific dictionary to use with a text. You can customize the patterns used in analysis and experiment to develop your knowledge of both DocuScope and rhetoric in general.

Creating a dictionary also creates a folder that houses a series of .TXT files that form the dictionary. You can edit any dictionary if you have access to this collection of .TXT files. .DAT files are not editable.
